# Pi Smart Display

## Purpose
Kiosk-style smart display optimized for a kitchen environment with a soft pink palette, ambient visuals, and web-based settings.

## Modules
- Clock + date
- Weather
- Meals
- News
- Calendar grid
- Context highlights

## Settings UI
Open `http://<pi-ip>:5173/settings` to configure:
- Feeds and calendars
- Location + timezone
- Module toggles
- Theme palette
- Performance mode and reduced motion
- Layout presets
- Optional admin token

## Deployment
See `docs/install-pi.md` for the one-command install flow and `docs/SPRINT_SECURITY_AND_PI_DEPLOY.md` for security and Pi hardening details.
